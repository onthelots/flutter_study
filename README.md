## Dart beginning

### 1. variables

- 변수, 상황에 따른 활용방식 (dynamic, final, late, const)
- Null Safety

### 2. data type

- dart 언어에서 사용되는 일반 데이터 타입
- 문자열 보간
- List 및 Collection 방식 (if, for)
- Map, Sets

### 3. functions

- 함수 선언방식
- 파라미터 활용방식 (Named, Defaults Value, Required Modified, Optioanl Positonal)

### 4. classes
- Class 선언 및 초기화 방식
- Constructors (+ 메서드)
- Enumeration
- 추상화 클래스(abstract) 및 상속방식 (extends)
- 생성자가 없는 클래스 (Mixins)
